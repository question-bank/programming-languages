## Print a triangle

Write a program to print on the output the following pattern:

```console
            C
          i   I
        s       s
      b           b
    e               e
  s                   s
t s e b s i C i s b e s t
```

<details>
  <summary>Answer in C++</summary>

  ```cpp
    #include <iostream>

    using namespace std;

    int main(){

        cout << "            C";
        cout << "          i   I";
        cout << "        s       s";
        cout << "      b           b";
        cout << "    e               e";
        cout << "  s                   s";
        cout << "t s e b s i C i s b e s t";

    }
  ```

</details>
