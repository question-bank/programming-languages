## Sum of Two Numbers

Write a program to print on the output the sum of two hard coded integer numbers (the numbers are up to you). For example, if you assigned 2 and 3, the output is:

**Output**:

```console
5
```

<details>
  <summary>Answer in C++</summary>

  ```cpp
    #include <iostream>

    using namespace std;

    int main(){

        int a;
        int b;
        int sum;

        a = 2;
        b = 3;

        sum = a + b;

        cout << sum << endl;
        
    }
  ```

</details>
